# CustomViewDraw

  The project of the drawing program for an android smartphone. You can draw with both a finger and a stylus. Program functions: drawing lines, choosing the thickness of the brush, back and forward buttons.
Clearing the screen and saving the picture in the phone memory. The program is built on the principles of pure architecture, with the exception of what is needed in the ViewModel in Presentation
add a repository dependency injection from the Data layer. The program uses 1 activity and 2 fragments. Implemented the ability to flip the screen without fear
of getting an exception, since the ViewModel is used.

  Проект программы - рисовки на андроид смартфон. Рисовать можно как пальцем, так и стилусом. Функции программы: рисование линий, выбор толщины кисти, кнопки назад и вперед.
Очистка экрана и сохранение рисунка в памяти телефона. Программа построена по принципам чистой архитектуры, за исключением того, что нужно еще во ViewModel в Presentation
слое добавить иъекцию зависимостей репозитория из Data слоя. В программе используется 1 активность и 2 фрагмента. Реализована возможность переворота экрана без опасения 
получить исключение, так как используется ViewModel.
